# **App Name**: Rebuy Tracker

## Core Features:

- User Authentication: Allow users to log in as either a 'Dealer' or a 'Player'.
- Initial Buy-in: Automatically provide a default buy-in for players when they join.
- Re-Buy Tracking: Enable Dealers to add and track re-buys for each player.
- Re-Buy Management: Allow players to request re-buys. Prevent players from removing re-buys after they are added by the Dealer.
- Dealer Privileges: Implement controls that allow only Dealers to add/remove re-buys for each player

## Style Guidelines:

- Primary color: Deep teal (#008080) to evoke a sense of trust and sophistication.
- Background color: Light teal (#E0F2F7) to keep the design clean and calm.
- Accent color: Mustard yellow (#FFDA63) to highlight important interactions.
- Body font: 'Inter', a sans-serif, suitable for displaying numeric values.
- Headline font: 'Space Grotesk', a sans-serif, for a tech-centric feel, complementing the data tracking aspect of the application.
- Use minimalistic icons to represent actions like 'add re-buy,' 'remove re-buy,' and different user roles.
- Clearly display user roles (Dealer, Player) with dedicated UI sections for each to avoid confusion.
